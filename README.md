# agent-instructions

This repository contains instructions, skills, and configurations for AI agents, particularly coding assistants. It serves as a collection of specialized capabilities that enhance agent functionality in various domains.

## Skills

- **code-simplifier**: Simplifies and refines code for clarity, consistency, and maintainability while preserving all functionality. Focuses on recently modified code unless instructed otherwise.
- **playwright-cli**: Automates browser interactions for web testing, form filling, screenshots, and data extraction. Use when the user needs to navigate websites, interact with web pages, fill forms, take screenshots, test web applications, or extract information from web pages.
